SquirrelPAK AVP - Beta 4
File Version 1.18.2+0.4.0
================================================================================

Check Ya Drawers Update 

In this update we fixed Issue #2 tindril pearls were not able to be used in some 
mods. Added a way to convert one to the other. Also changed the recipe for 
rotten flesh to leather to only use sticks and not a item frame.

Saw a remake of storage drawers called Functional Storage. It looks cool and the
fact that you do not have to have them touching from the controller looks 
awesome! So I have added that as well. We may remove Storage Drawers in later
if we think it's better.

Also added Ender Mail, a way to ship stuff around your world, thought that could
be a cool way to send stuff to your friends or to your new base 1000 blocks 
away.

Updated mods that had fixes

Last, in order to keep this mod pack going on and not have to rename after every 
update to Minecraft the pack version will start with the Minecraft version then
the pack version. This way we can release updated versions of the packs on the 
same mod pack page. 

Known Issues:
There is one known issue and that is with the options.txt file. There are issues
with the mods I usually use so until we get that fix I will ship a options 
file. Before updating to new versions if you make changes to your options file 
 make a backup copy first and replace it after updating.

The Money Update 

In this update we have included more server settings and tweaked 
lightmanscurrency config. We have enabled melting coins down to ingots in the 
coin press. Because of this, coins can be treated as resources. We have tweaked
the worth of each coin based on this, as well as tweaking recipes of the ATM and
coin press. Coin values are as follows: N=32, D=32e, E=2g, G=2i, I=100c, C=1c.
